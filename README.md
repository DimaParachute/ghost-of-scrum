# Ghost of Scrum Bot

Телеграм-бот — помощник для скрам-команды в рабочем чате.

## Возможности

- Напоминания о встречах: дейли по будням, еженедельные демо, двухнедельные ретро и т.п.
- Голосование за спринт (0–10) каждые 2 недели в пятницу второй недели спринта в 12:00 МСК. Считает среднее, показывает счётчик голосов, можно закрыть досрочно.
- Голосование по тестовой среде (0–5) сразу после, в 12:01 МСК. Голосуют зарегистрированные тестировщики, считается среднее.
- Случайный выбор фасилитатора дейли каждый будний день в 11:59 МСК. Если выбранный отказался — бот выбирает замену.
- В пятницу в конце спринта вместо случайного фасилитатора тегается зарегистрированный скрам-мастер (он ведёт планирование в понедельник).

Все события считаются от `SPRINT_START_DATE`. Дальше расписание тикает само каждые 14 дней — вручную начинать новый спринт не нужно.

## Локальный запуск (macOS / Linux)

Нужен Python 3.9+.

```bash
git clone <repo-url>
cd ghost-of-scrum-bot

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt    # либо см. список ниже
```

Если `requirements.txt` ещё нет, поставить можно так:
```bash
pip install "python-telegram-bot[job-queue]" APScheduler tzdata
pip freeze > requirements.txt
```

Создать `.env` в корне проекта:
```
TG_BOT_TOKEN=токен_от_BotFather
```

Запуск:
```bash
export $(grep -v '^#' .env | xargs)
python bot.py
```

## Настройки в `bot.py`

| Переменная | Что | По умолчанию |
|---|---|---|
| `TIMEZONE` | Часовой пояс расписания | `Europe/Moscow` |
| `TEAM_SIZE` | Размер команды (для авто-закрытия голосования) | `6` |
| `SPRINT_START_DATE` | Дата старта спринта (`YYYY-MM-DD`) | `2026-04-29` |
| `SPRINT_POLL_TIME` | Время голосования за спринт | `12:00` |
| `ENV_POLL_TIME` | Время голосования по тестовой среде | `12:01` |

Большинство параметров также меняется командами в чате (см. ниже).

## Команды бота

Это то же, что показывает `/start` в Telegram.

```
👋 Привет! Я скрам-помощник команды.

Что я умею:
1) Напоминать о встречах (дейли, ретро и др.)
2) Раз в 2 недели присылать голосование за спринт (0–10) и считать среднее

📌 Сначала зайди в рабочий чат и выполни /setchat — тогда я начну
   присылать туда голосование за спринт автоматически.

🛎 Команды напоминаний:
/daily ЧЧ:ММ Текст — каждый будний день
/weekly ДЕНЬ ЧЧ:ММ Текст — раз в неделю
/biweekly ДЕНЬ ЧЧ:ММ Текст — раз в 2 недели (отсчёт от sprint_start)
ДЕНЬ: mon, tue, wed, thu, fri, sat, sun
/list — список напоминаний этого чата
/remove ID — удалить напоминание по ID

📊 Голосование за спринт (пт 12:00 МСК, раз в 2 недели):
/startpoll — запустить голосование вручную прямо сейчас
/closepoll — досрочно закрыть голосование и показать среднее

🧪 Оценка тестовой среды (пт 12:01 МСК, раз в 2 недели, голосуют тестировщики, считается среднее):
/registertester — добавить себя в список тестировщиков
/unregistertester — убрать себя из списка
/addtester (reply, для админов) — добавить того, на чьё сообщение отвечаешь
/removetester (reply, для админов) — удалить того, на чьё сообщение отвечаешь
/testers — показать список тестировщиков
/startenvpoll — запустить голосование по тестовой среде вручную
/closeenvpoll — досрочно закрыть голосование и показать среднее

🎲 Случайный фасилитатор дейли (каждый будний день в 11:59 МСК):
/joindaily — добавить себя в список фасилитаторов
/leavedaily — убрать себя из списка
/addfacilitator (reply, для админов) — добавить того, на чьё сообщение отвечаешь
/removefacilitator (reply, для админов) — удалить того, на чьё сообщение отвечаешь
/dailymembers — показать список
/picknow — выбрать фасилитатора прямо сейчас (для теста)

🧭 Скрам-мастер (один на команду, ведёт планирование в пн):
В пятницу в конце спринта вместо случайного фасилитатора тегается скрам-мастер.
/registerscrum — зарегистрироваться скрам-мастером
/unregisterscrum — снять регистрацию
/scrum — показать зарегистрированного скрам-мастера
/setscrum (reply, для админов) — назначить того, на чьё сообщение отвечаешь
/unsetscrum (для админов) — снять текущего скрам-мастера

⚙️ Настройки:
/settings — показать текущие настройки
/setteamsize N — изменить размер команды
/setsprintstart YYYY-MM-DD — изменить дату начала спринта

Примеры:
  /daily 10:00 Дейли-стендап
  /weekly fri 16:00 Демо
  /biweekly thu 15:00 Ретро
```

## Порядок настройки в чате

1. Добавить бота в рабочий чат, сделать админом.
2. Выполнить `/setchat` — этот чат запомнится для авто-голосований.
3. Списки фасилитаторов и тестировщиков можно сформировать двумя способами:
   - **Самообслуживание:** каждый пишет `/joindaily` или `/registertester`.
   - **Админом чата:** в ответ (reply) на сообщение нужного человека выполни `/addfacilitator` или `/addtester`. Удалить — `/removefacilitator` / `/removetester` тем же способом.
5. Завести нужные напоминания: `/daily 10:00 Дейли-стендап`, `/biweekly thu 15:00 Ретро` и т.п.

## Известные ограничения

- Все хранилища (списки фасилитаторов, тестировщиков, активные голосования, конфиг) живут в памяти процесса. После перезапуска бота нужно заново выполнить `/setchat`, `/joindaily`, `/registertester`. Расписание авто-голосований и встроенных событий при этом восстанавливается из настроек в файле, а пользовательские напоминания (`/daily`, `/weekly`, `/biweekly`) теряются и заводятся заново.
- Тег пользователя кликабельный, если у него есть `username` или он хотя бы раз взаимодействовал с ботом (после `/joindaily` или `/registertester` — взаимодействовал).
